=========================================================
NOTES DE PRODUCTION GÉNÉRALES
=========================================================

Résolution et poids :
  – Toutes les photos : exporter en PNG optimisé (TinyPNG ou Squoosh).
  – Poids cible : < 150 Ko pour les avatars et icônes,
    < 350 Ko pour les photos de section (portrait, apropos),
    < 600 Ko pour le hero-banner.png.
  – Prévoir des versions @2x pour les écrans Retina.

Cohérence visuelle :
  – Les photos de personnes (hero, portrait, avatars) doivent partager
    une direction lumineuse cohérente : lumière naturelle chaude,
    légèrement latérale.
  – Les icônes Problèmes forment un set uniforme : même trait, même poids,
    même couleur rouge #C8372D.
  – Les icônes Offres forment un second set : même trait mais plus épais,
    couleur vert #0B5E38 (sauf icone-coaching-individuel.png en blanc).

Personnalisation :
  – Nom : remplacer "Élise Marchand" dans index.html et les métadonnées.
  – Couleurs : modifier --vert, --rouge, --ivoire dans :root{} de style.css.
  – Polices : remplacer l'import Google Fonts dans le <head> et mettre
    à jour --font-titre / --font-corps dans :root{}.

Ouvrir en local :
  – Placer index.html, style.css et le dossier assets/ dans le même
    répertoire, puis ouvrir index.html dans un navigateur.
  – Ou lancer : python3 -m http.server 8000
    puis ouvrir http://localhost:8000
